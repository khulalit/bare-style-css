Bare Style Css

Bare-style CSS is a lightweight CSS library that automatically applies styles to standard HTML elements. It eliminates the need for users to manually add CSS classes, enabling faster development with clean and simple HTML.

Bare-style CSS is perfect for situations where speed and simplicity are key. It's great for quick prototyping, educational projects, and static websites, allowing developers to focus on content rather than styling. It's also ideal for personal websites, internal tools, or dashboards, where minimal custom styling is needed.

Inline formatted text

Inline formatted text is used to apply specific styles or emphasis to a part of the text within a larger content block. This allows you to highlight or change the appearance of individual words or phrases without interrupting the flow of the surrounding text. Bare-style css provide easy and well themed formatted out of the box.

Below are some examples of inline formatted text elements:

  • Bold text: Use the <strong> tag to make text bold.
  • Italic text: The <em> tag is used for italicizing text.
  • Underlined text: The <u> tag applies an underline to the text.
  • Code snippet: Use the <code> tag to format inline code.
  • Highlighted text: The <mark> tag is used to highlight text.

Button Component Documentation

Default Button Style

The default button style follows the primary theme of the library with a background color of #3a6a59 and white text color.

Button Types

Buttons are customizable based on their type. Below are examples for each button type:

Primary Button

The primary button style uses the primary theme colors for actions.

Primary Outline Button

The outline version of the primary button has a transparent background with a border.

Danger Button

The danger button style is used for destructive actions, such as delete.

Success Button

The success button style indicates positive actions like success or confirmation.

Neutral Button

The neutral button style is for less important actions.
